# PyStockBot
This is a project to create a stock bot that can predict the stock price of a company using machine learning.



## Installation
```bash
pip install scikit-learn pandas numpy matplotlib xgboost yfinance seaborn autopep8 pandas-datareader keras tensorflow
```
